Help

If you want to edit an existing locale then click on the 'local file' link under 'Load' to load it up. If you're starting from scratch then go straight to the next step.

The important pages to edit are: identity, characters, miscellaneous and resources. Add what data you can to those pages. Click on a 'Save' button wherever you see it to make sure your changes stick.

When you've finished editing come back to this page and click on the 'local file' link under 'Save' to write the updated file to your computer.

Once you're happy with the saved file (which will be called xxx_Yyyy.xml), please email it to martin_raymond@scriptsource.org.

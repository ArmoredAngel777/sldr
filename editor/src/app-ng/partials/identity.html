Identity

The two most important fields on the Identity page are Language and Script. Please make sure there are values in both these fields. If the data you're entering is specific to how the language is used in a particular country then please enter a Region code as well. If not then you might want to set the Default Region to the code for the country where the most users of this language and script reside.

(Don't worry about the other fields on this page: you're not likely to need them.)

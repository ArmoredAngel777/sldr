Identity

The language component of a language tag is either the iso639-2 or iso639-3 language code (depending on which is shorter). It is entered as a lower case string.
The script component of a language tag is a 4 letter code with the first letter capitalised and the rest lower case. The value is taken from iso15924
The territory component of a language tag is either a 2 letter all caps code from ISO3166 or a 3 digit region code.

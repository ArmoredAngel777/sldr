Help

There are two separate lists of exemplar characters, both formatted in the same way. The main character list is the minimum set of alphabetic characters required to write this language with this script. The auxiliary character list, as the name implies, contains characters which are only used for writing loan words, such as foreign or technical words. No characters in the main list should be included in the auxiliary list. Where a writing system employs both upper and lower case characters, only the lower case characters should be included in these two character lists.

If combining marks are applied to a large number of base characters (as in Indic scripts), include each combining mark as a separate character. Where a combining mark is only used with relatively few base characters (for example tone marks, which are only applied to vowels), each combination should be listed as a separate character. Wherever a given combination is not a precomposed character within Unicode (i.e. it does not have a single codepoint), it should be included within braces. To illustrate, é does not require braces, but ɛ́ does: if in doubt, it doesn't harm to enclose the character in braces.

The index character list contains the 'shortcut' letters for jumping to sections of a sorted, indexed list (such as a telephone directory), and is formatted in the same way as the other lists. Where a writing system employs both upper and lower case characters, generally the upper case characters are used for the index character list.

The list of punctuation characters is the least important; if entered it should include characters used for all forms of punctuation, including denotation of questions and quotations.
